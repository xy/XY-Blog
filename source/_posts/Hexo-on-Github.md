# 在 Github 中搭建 Hexo 博客
## 部署步骤
``` bash
hexo clean
hexo generate
hexo deploy

```
### 本地调试
``` bash
hexo deploy -g  生成加部署
hexo server -g  生成加预览 (hexo s -g)
```
## 命令简写
``` bash
hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy
```
