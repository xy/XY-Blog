#Django
###Get Djano
三种安装方式:

* [Diango官网]()
* pip install Django==1.6.4
* [Git](https://github.com/django/django.git)

###Create Project

* django-admin.py startproject mysite

###Run Project
* python manage.py runserver
* Open [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
