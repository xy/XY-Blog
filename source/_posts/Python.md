#Python
##virtualenv
*	安装

~~~{.shell}
    sudo apt-get install python-virtualenv
~~~

或者

~~~{.shell}
    sudo pip install virtualenv
~~~

*	建立虚拟环境

~~~{.shell}
    virtualenv name
~~~

*	激活虚拟环境
进入创建的目录

~~~{.shell}
    source /bin/activate
~~~

##python-dev

~~~{.shell}
    sudo apt-get install python-dev
~~~
